# Advanced Pandas Sales Analysis

## Project Overview

This project demonstrates advanced data analysis using the Pandas library in Python. The analysis is performed on a sales dataset and covers data exploration, filtering, grouping, aggregation, pivot tables, and business insights.

The project was completed as part of a Data Analytics learning program and showcases practical usage of Pandas for real-world business reporting.

---

## Dataset

**File:** `Advanced_pandas_sales_dataset.csv`

The dataset contains sales transaction information including:

- Order ID
- Order Date
- Customer
- City
- Region
- Product
- Category
- Sales
- Quantity
- Discount
- Profit

### Q5. Product Performance Analysis

- Most popular product

### Q6. Multi-Level Grouping

### Q7. Pivot Table Creation
- Region vs Product sales matrix
- Highest laptop-selling region

### Q8. Profitability Analysis
- Average profit by category
- Most profitable category

### Q9. Discount Impact Study
- Average profit by discount level
- Relationship between discount and profit

### Q10. End-to-End Data Wrangling
- Sales filtering
- Grouping
- Aggregation
- Pivot table creation
- Business insights

---

## Key Insights

- East Region generated the highest revenue.
- Nitin was the highest contributing customer.
- Monitor was the most popular product by quantity sold.
- Office category had the highest average profit.
- Higher discounts generally resulted in lower profit margins.
- East Region dominated premium product sales such as laptops and phones.
